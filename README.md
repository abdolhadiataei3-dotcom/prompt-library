# Prompt Library

A free and open-source collection of high-quality prompts for AI assistants such as Claude, ChatGPT, Gemini, and GitHub Copilot.

## Features

- Programming prompts
- Writing prompts
- Study prompts
- Business prompts
- Community contributions welcome

## Project Goals

This project aims to provide well-organized, reusable prompts that help developers, students, writers, and professionals improve productivity with AI.
